# PlayPitch - Your Ultimate Sports Booking and Networking App

PlayPitch is an all-in-one mobile application designed to simplify your sports experience. Whether you're looking to book a turf for your favorite game (cricket, basketball, football, and more), organize a tournament, or connect with other sports enthusiasts, PlayPitch has got you covered. 

## 🚀 Features

### 🏟️ Book Sports Turfs
- Seamlessly book turfs for multiple sports like cricket, football, basketball, and more, all in one place.
- Browse through a list of available turfs in your locality with real-time availability.

### 🏆 Create Tournaments
- Organize your own tournaments for different sports.
- Invite your friends or other players to join the event and track its progress.
- Get tournament schedules and match results in real time.

### 🤝 Connect with Other Sports Enthusiasts
- Connect with fellow sports players, create a sports network, and stay updated on their availability for matches.
- Invite players to join events or casual games.

### 🎉 Event Management
- Easily invite other users to events.
- Manage event schedules, bookings, and other logistics seamlessly.
- Keep track of your upcoming events and tournaments with personalized notifications.

## 💡 Technology Stack

PlayPitch is built with cutting-edge technologies to provide the best user experience and performance. Here's a list of the main technologies used:

### Backend
- **Node.js**: JavaScript runtime built on Chrome's V8 JavaScript engine for server-side scripting.
- **Express.js**: Fast, unopinionated, and minimalist web framework for Node.js.
- **MySQL**: Relational database management system for storing user data, bookings, events, etc.
- **Sequelize**: Promise-based ORM for Node.js and MySQL, used to interact with the database.

### Frontend
- **React Native**: A framework for building native apps using React, making PlayPitch available on both iOS and Android platforms.

## 🛠️ Installation Guide

To get started with PlayPitch, follow these simple steps:

### 1. Clone the repository
```bash
git clone https://github.com/dsvasudev19/TurfSearchApp.git
```

### 2. Install Backend Dependencies
Go to the backend directory and install the necessary Node.js packages:
```bash

npm install
```

### 3. Set up the Database
Make sure you have MySQL installed on your machine. Create a database for PlayPitch and configure the database credentials in the `config/config.json` file.

### 4. Run the Backend
Start the backend server:
```bash
npm start
```

### 5. Install Frontend Dependencies
Go to the frontend directory and install React Native dependencies:
```bash
cd frontend
npm install
```

### 6. Run the App
To run the app on your mobile device/emulator:
```bash
npm run android   # For Android
npm run ios       # For iOS
```
