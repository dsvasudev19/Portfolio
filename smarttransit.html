SmartTransit

A Microservices-Based Smart Public Transportation System

System Architecture

Overview

SmartTransit System is a robust, scalable, and flexible microservices architecture designed for modern public transportation. It handles Carpooling, Bus Scheduling, User Management, Route Optimization, and integrates features like Live Bus Tracking using Ola Maps, along with Centralized Authentication to secure all services.

Key Features

Live Bus Tracking

Real-time bus location tracking and monitoring using Ola Maps integration.

Carpooling Service

Efficient carpooling management with route optimization and matching.

Route Management

Smart route optimization and planning for efficient transportation.

Notifications

Real-time alerts and updates for users about their travel status.

Payment Integration

Secure payment processing for bookings and services.

Centralized Authentication

Secure JWT-based authentication across all microservices.
