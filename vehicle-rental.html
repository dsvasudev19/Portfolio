Wheels On Demand

A Comprehensive Vehicle Rental Platform

Application Screenshots

Overview

Wheels On Demand is a comprehensive vehicle rental application developed using Spring Boot for the backend microservices, Angular for the client-side frontend, and React for the admin dashboard. The project provides a complete platform for managing vehicle rentals, with features including user management, vehicle management, feedback management, coupon management, and vendor management.

Key Features

User Management

Complete system for managing users, tracking rentals, and updating profiles.

Vehicle Management

Efficient system for adding, updating, and managing vehicle inventory.

Feedback Management

System to collect and manage user feedback for service improvement.

Coupon Management

Create and manage discount coupons for special offers.

Vendor Management

Complete system for managing vendors who provide vehicles.

JWT Authentication

Secure authentication system for user access control.
